# 额外软件仓库

## Teddysun
包括 Teddysun Shadowsocks Repository 和 Teddysun Linux Repository
```shell
bash teddysun.sh
```

## Caddy
RHEL 9 使用 Teddysun 源；Debian or Ubuntu 使用 cloudsmith 源
```shell
bash caddy.sh
```

## N.wtf
包含最新功能和额外模块的第三方 Nginx 源，只支持 Deb 系列
```shell
bash n.wtf.sh
```

## XanMod
默认集成优化的 Linux Kernel，只支持 Deb 系列
```shell
bash xanmod.sh
```
